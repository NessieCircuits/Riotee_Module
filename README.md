# Design Resources for the Riotee Module

The Riotee module integrates energy harvesting, energy storage, power management, non-volatile memory, a powerful Cortex-M4 processor, and a 2.4-GHz, BLE-compatible radio into a tiny module with the footprint of a postage stamp.

This repository hosts design resources to help users to integrate the Riotee module with their device. These resources include:
